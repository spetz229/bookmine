A3.2 The Existential Operator

Symbol

Alias

JavaScript

x?

typeof x != ’undefined’ && x !== null

func?()

if (typeof func === ’function’) {func()}

x?.y

x.y if x?

typeof x !== ’undefined’ && x !== null ? x.y : undefined

x ?= y

x = y unless x?

if (typeof x === ’undefined’ || x === null) {x=y}

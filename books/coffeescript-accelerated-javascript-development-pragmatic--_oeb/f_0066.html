Appendix 2
Ways of Running CoffeeScript

Although CoffeeScript’s ecosystem is young, there are a vast number of tools out there for compiling and running CoffeeScript code. We already covered the official command-line compiler in Section 1.3, ​Meet ’coffee’​; this appendix covers a few of the other options I recommend. For a more comprehensive list, visit http://github.com/jashkenas/coffee-script/wiki.

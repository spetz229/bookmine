A2.2 Running CoffeeScript in Your Web App

Wouldn’t it be nice if you could include CoffeeScript directly in your HTML without having to compile it to JavaScript first? Ah, but you can!

  <script type="text/coffeescript" >
  alert 'Wow, this CoffeeScript is right in your HTML!'
  </script>

The only catch is that you have to include a special version of the CoffeeScript compiler, which weighs in at a whopping 170 KB.[53]

Since it simplifies development, we used the browser-based compiler for our example project in Chapter 5, ​Web Interactivity with jQuery​. But unless you’re building your own CoffeeScript console, this approach isn’t recommended for production use. In addition to the sheer size of the compiler, all CoffeeScript files are loaded via Ajax after the compiler has been loaded.

The next few tools we’ll be looking at are aimed at bridging that gap between easy web development and efficient deployment.

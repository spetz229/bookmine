5.15. Practical Pointers and Further Reading

GHC already bundles a pretty-printing library, Text.PrettyPrint.HughesPJ. It provides the same basic API as our example but a much richer and more useful set of pretty-printing functions. We recommend using it, rather than writing your own.

John Hughes introduced the design of the ). The library was subsequently improved by Simon Peyton Jones, hence the name. Hughes's paper is long, but well worth reading for his discussion of how to design a library in Haskell.HughesPJ pretty printer "The Design of a Pretty-Printing library" (

In this chapter, our pretty-printing library is based on a simpler system described by Philip Wadler in "A prettier printer" (). His library was extended by Daan Leijen; this version is available for download from Hackage as wl-pprint. If you use the cabal command-line tool, you can download, build, and install it in one step with cabal install wl-pprint.

16.5. Parsec and MonadPlus

Parsec's GenParser monad is an instance of the MonadPlus typeclass that we introduced in Section 15.3. The value mzero represents a parse failure, while mplus combines two alternative parses into one, using (<|>):

-- file: ch16/ParsecPlus.hs
instance MonadPlus (GenParser tok st) where
    mzero = fail "mzero"
    mplus = (<|>)

Chapter 15
Global Commands

The :global command combines the power of Ex commands with Vim’s pattern matching abilities. We can use it to run an Ex command on each line that matches a specified pattern. Alongside the Dot Formula and macros, the :global command is one of Vim’s power tools for performing repetitive work efficiently.

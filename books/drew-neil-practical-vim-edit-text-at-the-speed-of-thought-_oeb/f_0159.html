Chapter 20
Find and Fix Typos with Vim’s Spell Checker

Vim’s spell checker makes it easy to find and correct spelling mistakes. In Tip 118, we’ll find out how to operate the spell checker from Normal mode, and in Tip 121, we’ll learn that it can also be operated from Insert mode.

Vim usually ships with a spell file for the English language only, but it’s easy to install others. Also, as we’ll see in Tip 119, we can switch between American and British English (beside others). And if a word is incorrectly marked as misspelled, adding it to a spell file is a simple matter, as we’ll see in Tip 120.

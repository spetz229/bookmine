Handling Events in Progress Bars with bind ()

In addition to event methods used through the options offered by the progressbar (options) method, jQuery UI allows us to manage these events using the bind () method.

To do this, jQuery UI includes the progressbarchange event, which has the same meaning as the options.change option.

Event Management on the Moved Elements with bind ()

In addition to event methods offered in the options of the draggable (options) method, jQuery UI allows us to manage these events using the bind () method (detailed in Table 10-9).

Table 10-9. Events created by jQuery UI

Event

Function

dragstart

Same meaning as options.start.

drag

Same meaning as options.drag.

dragstop

Same meaning as options.stop.

Event Management in Accordion Menus with bind ()

In addition to event methods used in the options of the accordion (options) method, jQuery UI allows us to manage these events using the bind () method. jQuery UI has created different events, listed in Table 3-5.

Table 3-5. jQuery UI events for managing accordion menus

Event

Function

accordionchange

Same meaning as options.change (see Table 3-3).

accordionchangestart

Same meaning as options.changestart (see Table 3-3).

These events allow you to perform treatments using the callback method provided by the bind (eventName, callback).
